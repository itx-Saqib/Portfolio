# Project Structure

```text
src/
├── animations/       # Reusable Framer Motion variants
├── assets/           # Images, SVGs
├── components/
│   ├── ui/           # Buttons, inputs, standard small components
│   ├── layout/       # Navbar, Footer, PageWrappers
│   ├── sections/     # Hero, About, Skills, Projects, Experience, Contact
│   ├── cards/        # ProjectCard, SkillCard, ExperienceCard
│   └── icons/        # Custom SVG icons
├── constants/        # Static data (skills list, projects list)
├── hooks/            # Custom React hooks
├── pages/            # Main route pages (Home, NotFound)
├── routes/           # Routing configuration
├── styles/           # Global CSS, Tailwind entry point
├── types/            # Definitions
└── utils/            # Helper functions
```
